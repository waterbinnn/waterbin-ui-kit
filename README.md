### Installation

`yarn add @waterbin/ui-kit`

✅ Requirements

```
- React ≥ 18.x
- ReactDom ≥ 18.x

NextJs 에서 사용시 'use client' 작성 필요
```

### Dependencies

```
- React
- Typescript
- Vite
- Panda CSS
- Storybook
```
